# Palomino Coastal Data Analysis - Proof of Concept
**Autor:** [Tu Nombre]
**Propósito:** Demostración de capacidades analíticas para la evaluación de variables oceanográficas (corrientes, oleaje) y modelado de erosión costera en el corregimiento de Palomino, La Guajira.

## Tecnologías Utilizadas
* Python (Pandas, NumPy)
* Matplotlib / Seaborn (Visualización de datos)
* Open-Meteo Marine API (Ingesta de datos oceanográficos en tiempo real)

## Metodología
Este proyecto extrae pronósticos de oleaje y velocidad de corrientes y los cruza con modelos estadísticos de retroceso costero para facilitar la toma de decisiones estratégicas y la gestión de riesgos en infraestructuras turísticas.
